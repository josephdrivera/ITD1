3.6 Code Exercise 06: HTML Forms

Overview:

For this assignment, you will be solving 2 different problems in the same HTML & JS files. This will help strengthen your understanding of HTML Forms.

Objectives:

Successful completion of this activity will show that you can do the following:

  • Determine which input types are needed for what information.
  • Create interactive HTML pages using forms and buttons.
  • Formulate conditional statements for dynamic decision-making.
  • Create logic to resolve a single solution from a selection of possible solutions.
